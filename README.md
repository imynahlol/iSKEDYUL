# iSKEDYUL

> iSKEDYUL: Intelligent Scheduling and Classroom Reservation System for PUP Lopez
> Computer Programming 1 | BSITLQ1-1

# Project Overview
iSKEDYUL is a C++ console-based prototype scheduling and classroom reservation system developed for the Polytechnic University of the Philippines (PUP) – Lopez Campus. The system is intended to replace the existing manual and informal classroom booking processes currently in use. It provides a centralized and transparent digital platform that allows authorized users, including faculty members, administrators, and student organization representatives, to check real-time classroom availability, reserve rooms for their preferred dates and time slots, view existing schedules, and manage or cancel reservations. By integrating all classroom scheduling activities into a single accessible system, iSKEDYUL aims to prevent double-booking, reduce scheduling conflicts, and promote the efficient use of classroom facilities, thereby supporting a more organized and effective academic environment at PUP Lopez.
